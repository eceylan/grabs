## Grabs Install

Node.js and npm must be installed.

Steps:

1. `sudo npm install -g grunt-cli`
2. `git clone git@github.com:sercaneraslan/Grabs.git`
3. `npm install`
4. `grunt bower:install`
5. `grunt`

Check the address: http://localhost:9000/
