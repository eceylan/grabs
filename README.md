## Grabs Install

Node.js, Npm and Grunt must be installed.

Steps:

1. `git clone git@github.com:sercaneraslan/grabs.git`
2. `npm install`
3. `grunt bower:install`
4. `grunt`

Check the address: http://localhost:9000/
